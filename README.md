# pygame-walking-demo

A simple side-scrolling Python game built with **Pygame**.  
This project demonstrates basic character animation, background movement, and game-loop structure.  
The game is still in early development, and new features will be added over time.

---

##Features
- Character walking animation  
- Parallax-style moving background  
- Custom window title ("Esen The Best")  
- Basic rendering system  
- Organized asset structure
